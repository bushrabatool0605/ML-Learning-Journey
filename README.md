 Machine Learning Journey Portfolio

Welcome to my Machine Learning and Data Science journey! This repository contains my 8-month progression from fundamental data analysis to building and deploying complex Deep Learning models and interactive web applications.

 Project Structure

 01-Regression-Models: Predictive modeling using Linear and Multiple Regression.
 02-Classification-Models: Classification projects including Iris, Titanic, and Heart Disease prediction.
 03-Deep-Learning-TensorFlow: Neural Networks implementation for complex classification tasks.
 04-Heart-Disease-Streamlit-App: An interactive web app deployed using Streamlit.

 Tech Stack
 Languages: Python
 Libraries: Pandas, NumPy, Scikit-Learn, TensorFlow, Keras
 Visualization: Matplotlib, Seaborn
 Deployment: Streamlit

 Learning Highlights
Throughout this journey, I have focused on:
1.  Data Preprocessing: Cleaning, feature engineering, and normalization.
2.  Model Building: Implementing algorithms from scratch and using libraries.
3.  Deployment: Turning models into user-friendly web apps.
